# WeatherControl

## Overview
WeatherControl is a cross-platform weather application built with .NET MAUI.  
The project provides a clean interface for retrieving current weather conditions by ZIP code or by selecting a major city, then presents temperature and atmospheric details in a structured view.

## Technologies Used
- C#
- .NET MAUI (Single Project structure)
- XAML for UI layout and visual design
- `HttpClient` for API communication
- OpenWeather API for live weather data
- Newtonsoft.Json (`JObject`) for JSON parsing
- Visual Studio Code and .NET CLI for development workflow

## Project Focus
- Modern, professional user interface design
- Responsive page structure and readable data presentation
- Reliable async networking and error handling for weather lookups
